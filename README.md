# Farm-management
Web application that allows you to keep documentation of operations
performed on the agricultural areas.
#### Possible operations
* create new user
* update user details in user cockpit
* add a new field
* update/delete a field
* add new operations for the selected field
* done selected task for selected field
* update/delete a field operation
* user management by admin (Admin account added to the database, login:admin, password:admin)
#### Prerequisites
* Java 8 or newer
* Your preferred IDEA (f.e IntelliJ IDEA)
#### Used technologies and tools
* Spring: MVC,Data,Security
* HTML 5
* CSS 3
* JavaScript
* H2
* Maven
